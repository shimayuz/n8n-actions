# ワークフローディレクトリ

このディレクトリには、すべてのn8nワークフローJSONファイルが含まれています。

## 構造

- ワークフローJSONファイルはこのディレクトリに直接配置してください
- サンプルワークフローは `samples/` サブディレクトリを使用してください
- 各ワークフローは有効なn8n JSONエクスポートである必要があります

## ガイドライン

1. **1ファイル1ワークフロー**: 各JSONファイルには正確に1つのワークフローを含める必要があります
2. **説明的な名前**: 明確で説明的なファイル名を使用してください
3. **有効なJSON**: すべてのファイルが有効なJSON形式であることを確認してください
4. **認証情報なし**: コミット前にハードコードされた認証情報を削除してください

## ファイル命名規則

- `customer-onboarding.json` ✓
- `daily-backup-automation.json` ✓
- `workflow1.json` ✗ (説明的でない)
- `test.json` ✗ (説明的でない)
